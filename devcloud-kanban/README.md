# devcloud kanban app

> devcloud kanban app

## 构建命令

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build -- --report

# run lint
npm run lint
npm run lint -- --fix

# run unit tests
npm run test:unit

# run e2e tests
npm run test:e2e
```
